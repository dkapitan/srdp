Narrative: A Data Station is a specialization of a EIRA Shared Platform. It aims to move towards standardization set of application components: - Data Persistence: data is available in a Data Station for use by licensed third-parties - Data Extraction, Transformation and Loading (ETL): the Data Station may contain a set of predefined application components to facilitate population of the Data Station with data from internal datawarehouse of the Data Holder -
