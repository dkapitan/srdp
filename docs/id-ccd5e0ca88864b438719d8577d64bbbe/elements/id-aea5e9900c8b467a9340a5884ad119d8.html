Analytics-to-the-data ()
(en) Perform an analysis at the location of the data. The results from the analyses are returned without requiring direct access to the analyzed data (nl) Het uitvoeren van een analyse op de locatie van de data. Het analyseresultaat wordt teruggezonden zonder dat directe toegang tot de geanalyseerde data nodig is.
